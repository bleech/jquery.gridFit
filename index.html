gridFit jQuery Plugin by bleech

The gridFit plugin enables you to let any block fit exactly into a custom grid that you can define at initialisation.

View the Project on GitHub bleech/jquery.gridFit

Features

  • target any block element
  • define a grid by setting the height and width of one unit
  • support for nested grid elements
  • margin and padding aware

Usage

Include the plugin file and then simply select one or more elements and execute gridFit.

Example for an element with the class example and a desired unit size of 20px in width and 40px in height:

$('.example').gridFit( 20, 40 );
